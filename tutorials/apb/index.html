Beginner's APB

APB (Athefre's pair & block) is a highlight algorithmic 3x3 method. This tutorial is going to teach a subset with slightly less algorithms (x as opposed to x, x% reduction)

This tutorial assumes that you at least know beginner's CFOP with intuitive F2l and 4LLL

Steps

  • 2. DF and DB edges
  • 3. Pair creation
  • 4. EOPair

    • 4b. EO
  • 5. LXS
    • 5a. DR edge
    • 5b. Last F2l Pair
  • 6. Last layer
    • 6a. OCLL
    • 6b. PLL (1 or 2 look)
